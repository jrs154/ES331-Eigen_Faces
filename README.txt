Run the code violajones.py to see the output of face detection and its false positive and false negative rates.
To check for any other image, pass its address into cv.imread() function in the 2nd half of code.

Run the code eigenfaces.py to see the output of face classification system for positive and negative samples.
To check for any other face, pass its address in the cv.imread() function in the 2nd half of code.

Use the file resize.py to convert any image to 250x250 pixels size so that it can be used by eigenfaces.py code.
Pass the address of image into cv.imread() function. 

'discarded' folder can be deleted, rest of the folders should be left unchanged.
